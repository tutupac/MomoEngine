# Momo Engine

2D engine dedicated to my momo so she can make games easily when shes allowed to touch a computer but like also for the people. No matter how much better or shittier it becomes it will be completely free.

---

## Featuring

- C++
- SFML
- AngelScript
- Dear ImGui (for editor)
- DragonBones (temporary)
- Cmake
- ECS
- Scene Management
- Events
- A* Pathfinding
- Basic Collision
- Game State
- Input
- Time
- Camera
- UI system (in game)
- Debugga

---


## Installation Guide

Install, Extract, Open Installer, Follow Setup, Voila (youll get the option to install dragonbones in it too)
